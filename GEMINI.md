# Krishna Library CLI Guidance

## Architectural Rules
- Prioritize functional components with React Hooks.
- Use Tailwind CSS and glassmorphism styling (`glass-pane` class).
- Animations should use Framer Motion when applicable for premium UI feel.
- Supabase is the backend data store.

## Styling Rules
- Adhere to the `font-manrope` and `font-body-md` typography setup.
- Stick to the existing color variables (e.g., `text-primary`, `bg-surface-container`).
- Ensure all forms have proper contrast and visual feedback on submission.
